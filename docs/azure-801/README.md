## Exam at a Glance

- **Duration:** 90–120 minutes
- **Questions:** 40–50 including case studies + hands-on labs
- **Passing Score:** ~700/1000 (~70%)
- **Cost:** ~$165 USD
- **Prerequisites:** AZ-900 recommended, not required

## Topic Weights (approximate)

| Domain | Weight |
|---|---|
| Manage Azure identities and governance | 20–25% |
| Implement and manage storage | 15–20% |
| Deploy and manage Azure compute | 20–25% |
| Implement and manage virtual networking | 15–20% |
| Monitor and maintain Azure resources | 10–15% |

{: .tip }
The exam includes **hands-on lab tasks** in the Azure portal. Practice creating and configuring resources — reading alone isn't enough.
