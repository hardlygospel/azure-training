# Azure Administrator (AZ-801) 🛠️

The next step after Azure Fundamentals. Covers **hands-on management** of Azure infrastructure.

## Chapters

1. [Identity & Access Management](./01-identity-access.md)
2. [Virtual Networking](./02-virtual-networking.md)
3. [Compute Management](./03-compute-management.md)
4. [Storage & Backup](./04-storage-backup.md)
5. [Monitoring & Compliance](./05-monitoring-compliance.md)

## Exam Format

- **Duration:** 90-120 minutes
- **Questions:** 40-50 (case studies, multiple choice, labs)
- **Passing Score:** ~70%
- **Cost:** $165
- **Prerequisites:** Azure Fundamentals recommended (not required)

## What You'll Do

- Create and manage VMs
- Configure networking (VNets, subnets, routing)
- Manage identities and permissions
- Back up and recover data
- Monitor resources and troubleshoot issues

## Key Differences from AZ-900

| AZ-900 | AZ-801 |
|--------|--------|
| Know what services exist | How to configure and manage them |
| Understand concepts | Hands-on, practical labs |
| Beginner | Intermediate/Advanced |

---

**Tip:** Expect hands-on labs where you'll actually create resources. Practice in the Azure Portal!
