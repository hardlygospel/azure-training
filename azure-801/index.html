    Azure Administrator (AZ-801)¶

    Hands-on Azure administration. You will configure, manage, and troubleshoot real Azure resources — including labs during the exam.

    Chapters¶

    # Topic What it covers
    1 Identity & Access AAD, RBAC, MFA, managed identities, Conditional Access
    2 Virtual Networking VNets, subnets, NSGs, routing, VPN, ExpressRoute
    3 Compute VMs, scale sets, containers, AKS, extensions
    4 Storage & Backup Blob, Files, backup vault, disaster recovery
    5 Monitoring & Compliance Azure Monitor, KQL, alerts, Policy, governance

    Exam at a Glance¶

    • Duration: 90–120 minutes
    • Questions: 40–50 including case studies + hands-on labs
    • Passing Score: ~700/1000 (~70%)
    • Cost: ~$165 USD
    • Prerequisites: AZ-900 recommended, not required

    Topic Weights (approximate)¶

    Domain Weight
    Manage Azure identities and governance 20–25%
    Implement and manage storage 15–20%
    Deploy and manage Azure compute 20–25%
    Implement and manage virtual networking 15–20%
    Monitor and maintain Azure resources 10–15%

    {: .tip } The exam includes hands-on lab tasks in the Azure portal. Practice creating and configuring resources — reading alone isn't enough.
